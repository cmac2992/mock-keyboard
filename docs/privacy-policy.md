# Privacy Policy

Last updated: March 9, 2026

Mock Keyboard is a developer tool for Chrome DevTools.

## What The Extension Does

Mock Keyboard helps developers simulate a mobile software keyboard while testing responsive interfaces in Chrome DevTools.

## Data Handling

Mock Keyboard stores extension settings locally in Chrome storage so the DevTools panel and the currently inspected tab can stay in sync.

These settings may include:

- whether the simulator is enabled
- the selected keyboard preset
- visibility mode
- debug preferences

Mock Keyboard does not send browsing data, page content, or extension settings to any remote server.

Mock Keyboard does not sell user data.

Mock Keyboard does not use data for advertising, marketing, or profiling.

Mock Keyboard does not require an account or sign-in.

## Page Access

Mock Keyboard injects code into the page currently being tested in Chrome DevTools so it can:

- render the mock keyboard overlay
- simulate viewport behavior
- expose keyboard-related viewport changes to the page for development purposes

This page access is used only to provide the extension’s core functionality.

## Third Parties

Mock Keyboard does not share user data with third parties.

## Changes

If the extension’s data-handling behavior changes in the future, this policy will be updated.
